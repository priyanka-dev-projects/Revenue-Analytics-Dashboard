# Revenue Analytics Dashboard

## Overview
This Power BI dashboard analyzes transaction data from the Transactions_Master table.

## Key Metrics
- Total Transactions
- Total GMV
- ERP Revenue
- Edviron Net Revenue
- Pending Exposure
- Success Rate %

## Features
- Interactive slicers (Date, Gateway, Payment Method, Partner)
- KPI cards
- Revenue trend analysis
- Dynamic filtering

## Tools Used
- Power BI
- DAX
- Power Query
